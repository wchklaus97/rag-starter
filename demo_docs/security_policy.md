Security Policy Summary

All production systems require multi-factor authentication.
Shared accounts are not allowed.

Sensitive company data includes customer records, pricing, unreleased product plans, and employee personal information.
Sensitive data must only be stored in approved systems.

Incident reporting:
- Report suspected phishing or credential leaks to security@acme.example within 30 minutes.
- Report lost or stolen devices to IT and Security immediately.
- Security will start triage within 1 hour for critical incidents.

Access reviews happen every quarter.
Contractors must use time-limited accounts that expire automatically at the end of the engagement.
